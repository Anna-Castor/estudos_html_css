Nessa frase temos um termo em destaqueusando a tag B (não semântica)

Já nessa frase temos um outro termo em destaqueusando a tag STRONG (semântico)

Itálico/Ênfase

Nessa frase temos um termo em itálico usando a tag I (não semântica)

Já nessa outra frase temos outro termo em itálico usando a tag EM (semântica)

Texto marcado

Podemos criar também um texto marcado usando a tag MARK no HTML.

Para mudar a cor dessa marcação só usando o CSS style. Posso ainda deixar padrão a cor que quero para todas as marks, mas nessa caso eu teria que criar a marcação lá na HEAD e ele aplicaria a todos os marks

Texto Grande e Pequeno

Estamos criando um texto grande usando a tag BIG (OBSOLETA)e um texto pequeno usando a tag SMALL.

Texto Deletado

Podemos marcar um texto como excluído para indicar que ele deve ser lido, mas não considerado usando tag DEL

Texto Inserido/Sublinhado

Podemos marcar um texto como inserido para dar uma ênfase e indicar que ele foi adicionado depois usando tag INS.
