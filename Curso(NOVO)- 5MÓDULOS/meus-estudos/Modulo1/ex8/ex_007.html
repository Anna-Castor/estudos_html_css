Principais Formatações

Negrito/Destaque

Nessa frase temos um termo em negrito usando a tag B (não semântica)

E nessa frase temos um termo em negrito usando a tag STRONG, mais adequada e semântica.

Itálico/Ênfase

Aqui temos um termo em itálico usando a tag antiga I, não semântica

Mas aqui temos outro termo em itálico usando a tag EM (emphasis), que é semântica

Texto marcado

Podemos criar um texto marcado muito legal usando tag MARK

Podemos ainda personalizar a cor dessa marcação usando CSS

Texto Grande e Texto Pequeno

Nesse caso a tag BIG está em desuso, mas a tag SMALL ainda usa

Texto Deletado

Para texto deletado usamos a tag DEL, indicando que não deve ser considerado

Texto Inserido

Podemos marcar um texto como inserido com a tag INS para dar ênfase e indicar que ele foi adicionado depois

Texto Sobrescrito

Para inserir coisas do tipo x2 = 9, com tag SUP

Texto Subscrito

Para inserir coisas do tipo H20, com tag SUB

O comando document.getElementById('teste') é escrito em linguagem JavaScrip usando a tag CODE

O código abaixo usa a tag PRE e a tag CODE:

            
        num = int(input('Digite um número'))
        if num % 2 == 0:

            print(f'O número {num} é ÍMPAR')
        print('Fim do programa')
